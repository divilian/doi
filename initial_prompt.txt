3) another menu option is to perform a DOI lookup based on the search information given thus far. when the user chooses this, call the function you just wrote, and present to the user a numbered list of DOI matches, and give them the option to select one. when they select one, call each of the other three functions, and present that information to the user in a pretty way. the DOI list should then be printed again, and the user returned to the DOI-selection prompt. if they type "q" to this DOI prompt instead of a number, return to the first menu, with their search terms still intact.
