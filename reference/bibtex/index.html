    smeli.bibtex¶

    BibTeX parsing, pretty-printing, and Smeli fallback generation.

    BibTeX parsing, pretty-printing, and generation.

    Smeli prefers doi.org-provided BibTeX when it is available for a DOI. These helpers support parsing and displaying that BibTeX, and generating a conservative fallback BibTeX-like entry from Smeli candidate metadata.

    candidate_to_bibtex ¶

    candidate_to_bibtex(candidate: dict[str, Any]) -> str

    Generate a conservative BibTeX-like entry from candidate metadata.

    Parameters:

    Name Type Description Default
    candidate dict[str, Any]

    A Smeli candidate dictionary.

    		 required

    Returns:

    Name Type Description
    str str

    A BibTeX-like string using available fields such as title, authors, year, venue, DOI, arXiv ID, and URL.
    Notes

    This is a fallback formatter, not a replacement for publisher- or resolver-provided BibTeX. Prefer smeli.sources.get_bibtex_from_doi() when a DOI is available and doi.org content negotiation succeeds.

    print_bibtex ¶

    print_bibtex(bibtex: str) -> None

    Print a BibTeX entry in a friendlier field-by-field format.

    Parameters:

    Name Type Description Default
    bibtex str

    Text containing one BibTeX entry.

    		 required

    Returns:

    Name Type Description
    None None

    Output is written to standard output.
    Notes

    The raw BibTeX is printed after the field-by-field display so it can still be copied into BibTeX-aware tools.

    parse_bibtex_entry ¶

    parse_bibtex_entry(bibtex: str) -> dict[str, Any] | None

    Parse a single BibTeX entry into a simple dictionary.

    Parameters:

    Name Type Description Default
    bibtex str

    Text containing one BibTeX entry.

    		 required

    Returns:

    Type Description
    dict[str, Any] | None

    dict[str, Any] | None: A dictionary with entry_type, cite_key, and fields keys, or None if the text does not look like a normal single BibTeX entry.
    Notes

    This parser is intentionally small. It handles ordinary BibTeX returned by DOI content negotiation, but it is not a full BibTeX parser.

    make_cite_key ¶

    make_cite_key(candidate: dict[str, Any]) -> str

    Create a compact citation key from first-author surname and year.

    Parameters:

    Name Type Description Default
    candidate dict[str, Any]

    A Smeli candidate dictionary.

    		 required

    Returns:

    Name Type Description
    str str

    A lower-case key in Kurrent/Smeli style, such as "davies2011" or "starnini2025". Missing authors fall back to "paper" and missing years fall back to "nd".
